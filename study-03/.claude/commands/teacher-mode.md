선생님 모드 통합 리포트를 실행해줘. 아래 단계를 **순서대로** 수행하고, 각 단계 시작 시 "━━━ 단계 N/5: {단계명} ━━━" 형식으로 구분선을 출력해줘.

## 사전 준비: 데이터 파일 확인

`teacher-data/rankings.json` 파일 존재 여부를 먼저 확인해줘.

파일이 없으면 다음 안내를 출력하고 **즉시 종료**해:

```
======================================================
  선생님 모드를 시작하려면 학생 데이터가 필요합니다.
======================================================

1. 퀴즈 앱이 열린 브라우저에서 F12 → 콘솔 탭 열기
2. 아래 명령어 입력:
     copy(localStorage.getItem('quiz-storage'))
3. 복사된 내용을 teacher-data/rankings.json 에 저장
4. 또는 명령어로 직접 전달:
     /teacher-import {JSON 붙여넣기}

준비가 되면 /teacher-mode 를 다시 실행하세요.
======================================================
```

파일이 있으면 → `exportedAt`과 `totalCount`를 읽어서 "데이터 로드 완료: {n}개 기록 / 마지막 내보내기: {날짜시간}" 출력 후 진행.

## 단계 4/5: 주의 학생 종합

단계 1~3의 결과를 종합해서 선생님이 특별히 관심을 가져야 할 학생을 아래 기준으로 선정해줘.

선정 기준 (하나 이상 해당 시 포함):
- 최근 7일 미활동
- 2개 이상 카테고리에서 반 평균 미달
- 성적 추이가 📉 하락
- 종합 점수가 반 평균의 60% 미만

```
⚠️ 주의 학생 ({n}명)

  {닉네임}
    사유: {해당 기준 나열}
    마지막 플레이: {날짜}
    권장 조치: {개인 면담 / 취약 카테고리 집중 안내 / 재참여 독려 등}

  ...
주의 학생이 없으면: "✅ 전체 학생 정상 참여 중 — 특별 조치 불필요"
```

## 단계 5/5: 선생님 종합 리포트

모든 분석 결과를 한 페이지 요약으로 출력해줘.

```
╔══════════════════════════════════════════════════╗
║          선생님 모드 — 종합 리포트               ║
║          분석 일시: {오늘 날짜 YYYY-MM-DD}        ║
╠══════════════════════════════════════════════════╣
║ 참여 학생: {n}명 / 총 플레이: {n}회              ║
╠══════════════════════════════════════════════════╣
║ [성적 현황]                                      ║
║  1위: {닉네임} ({n}점)                           ║
║  반 평균: {n}점                                  ║
║  최고 달성률 카테고리: {카테고리} ({n}%)          ║
║                                                  ║
║ [취약 포인트]                                    ║
║  반 전체 취약: {카테고리명} (달성률 {n}%)         ║
║  개인 지도 필요: {n}명                           ║
║                                                  ║
║ [성장 현황]                                      ║
║  성장 {n}명 / 유지 {n}명 / 하락 {n}명           ║
║  이번 주 신규 참여: {n}명                        ║
║                                                  ║
║ [이번 주 권장 액션]                              ║
║  1. {가장 시급한 조치}                           ║
║  2. {두 번째 조치}                               ║
║  3. {세 번째 조치}                               ║
╚══════════════════════════════════════════════════╝

개별 심층 분석이 필요하면:
  /teacher-compare {닉네임1} {닉네임2}  — 학생 비교
  /teacher-progress {닉네임}            — 개인 추이
  /teacher-weak {닉네임}                — 개인 취약점
```
